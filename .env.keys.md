# MMFV вЂ” API Keys Reference Guide

## How to add a new API key
1. Add placeholder to `.env.example` (root)
2. Add real value to `backend/.env` (never commit)
3. Add real value to `src/.env.local` (never commit)
4. Add variable name to this file under the right section

## Security rules
- `RAZORPAY_KEY_SECRET` вЂ” **never** exposed to client. Pass `RAZORPAY_KEY_ID` to frontend via a server-side API route only.
- `SUPABASE_SERVICE_KEY` вЂ” **never** in any `NEXT_PUBLIC_*` variable
- `ANTHROPIC_API_KEY`, `JWT_SECRET`, `NEXTAUTH_SECRET` вЂ” backend / server-side only
- `NEXT_PUBLIC_*` вЂ” safe to expose; anon/public values only

## Payment processor
Razorpay ONLY. `RAZORPAY_KEY_ID` + `RAZORPAY_KEY_SECRET`. No Stripe.

## Email provider
Resend.com ONLY via `RESEND_API_KEY`. No AWS SES, no SMTP.
