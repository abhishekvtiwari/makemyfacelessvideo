# 🔑 MMFV — API Keys Reference Guide

## How to add a new API key
1. Add placeholder to backend/.env.example
2. Add placeholder to src/.env.local.example  
3. Add real value to backend/.env (never commit)
4. Add real value to src/.env.local (never commit)
5. Add variable name to this file under the right section
6. Update CLAUDE.md environment variables section

## Where to get each key

| Key | Where to get it | Used in |
|-----|----------------|---------|
| ANTHROPIC_API_KEY | console.anthropic.com → API Keys | Backend |
| RESEND_API_KEY | resend.com → API Keys | Backend |
| GOOGLE_CLIENT_ID | console.cloud.google.com | Both |
| SUPABASE_URL | supabase.com → Settings → API | Both |
| SUPABASE_SERVICE_KEY | supabase.com → Settings → API → Secret | Backend only |
| NEXT_PUBLIC_SUPABASE_ANON_KEY | supabase.com → Settings → API → Publishable | Frontend |
| ELEVENLABS_API_KEY | elevenlabs.io → Profile → API Keys | Backend |
| PEXELS_API_KEY | pexels.com/api | Backend |
| RAZORPAY_KEY_ID | dashboard.razorpay.com → Settings → API | Backend |
| STRIPE_SECRET_KEY | dashboard.stripe.com → Developers → API Keys | Backend |
| R2_ACCOUNT_ID | dash.cloudflare.com → R2 | Backend |
| JWT_SECRET | Generate: openssl rand -base64 32 | Backend |
