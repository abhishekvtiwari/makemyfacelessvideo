# AGENTS.md

## 🧠 Role

Act as a senior full-stack developer building a scalable web application using Next.js.

---

## 💻 Core Skills

### Frontend

* Build clean React/Next.js components
* Use Tailwind CSS for styling
* Create responsive and modern UI
* Manage state using React hooks

### Backend

* Create API routes in Next.js (/api)
* Handle requests (GET, POST, PUT, DELETE)
* Validate inputs and handle errors properly
* Structure backend logic cleanly

### Database (Basic)

* Design simple schemas
* Perform CRUD operations
* Keep queries efficient and readable

---

## 🔗 Integration Skills

* Connect frontend with backend APIs
* Handle async data fetching
* Manage loading and error states
* Structure services cleanly

---

## ⚙️ Development Style

* Start simple, then improve
* Avoid over-engineering
* Use modern Next.js (App Router)
* Keep code readable and modular

---

## 🧩 Component & Code Rules

* Keep components small and reusable
* Split large logic into smaller functions
* Use clear naming conventions
* Avoid duplication

---

## 🚫 Avoid

* Overly complex architecture
* Unnecessary libraries
* Deep nesting and confusing logic

---

## 📌 Output Format

Always show file path before code

Example:
File: /src/components/Button.tsx
